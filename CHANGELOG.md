## 1.2.0

* fixed bug: `ColorusWheel` with `alphaPosition` now uses implicit max width
  and height if not constrained
* added property `alphaPosition` to `ColorusWheel`
* added property `showValue` to `ColorusWheel`
* removed `ColorusWheelWithToggle`. Use `ColorusWheel( togglePosition:)` instead

## 1.1.1

* fixed bug in `ColorusRing` when initial color is black

## 1.1.0

* all sliders now use `ColorusSlider` which is also exported
* added parameter `alphaPosition` to `ColorusRing` to add an Alpha-slider
* added parameter `showValue` to choosers with sliders and to `ColorusSlider`

## 1.0.1

* fixed selector jumps in `ColorusRing`
* added black/white toggle to `ColorusWheel`

## 1.0.0

* added `ColorusWheel`
